CanvasFlow

Connect Claude Desktop to your Canvas LMS data through a Chrome extension and MCP server

What is CanvasFlow?

CanvasFlow enables you to access your Canvas LMS courses, assignments, and grades directly from Claude Desktop through natural conversation. Simply install the Chrome extension and MCP server, then ask Claude about your coursework.

List Courses

View all your active Canvas courses with enrollment details and course information.

Track Assignments

Check upcoming assignments with due dates, submission status, and grading information.

View Grades

Access your grades and submission history for completed assignments.

Natural Queries

Ask Claude in plain language about your courses, deadlines, and academic progress.

Installation

Step 1: Install Chrome Extension

  1. Download the extension from the Chrome Web Store (coming soon) or load from source
  2. Configure your Canvas URL using the Auto-Detect feature
  3. Click "Refresh Canvas Data" to sync your information

Step 2: Install MCP Server for Claude Desktop

Choose your preferred installation method:

Option A: Drag-and-Drop (Easiest)
  1. Download canvasflow-native-host.dxt
  2. Open Claude Desktop → Extensions
  3. Drag the .dxt file into the extensions area
Option B: Chrome Extension Integration
  1. Download canvasflow-chrome-extension.zip
  2. Extract and run the installer:
    • macOS/Linux: ./install.sh
    • Windows: install.bat
  3. Enter your Chrome extension ID when prompted

Step 3: Start Using CanvasFlow

  1. Restart Chrome and Claude Desktop
  2. In Claude Desktop, ask: "What are my Canvas courses?"
  3. Enjoy seamless access to your Canvas data!

Requirements
