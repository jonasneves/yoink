What is CanvasFlow?

CanvasFlow is a Chrome extension that transforms how you manage your Canvas LMS coursework. View assignments instantly, get AI-powered insights with Claude Extended Thinking, and generate optimized weekly study schedules—all without needing a Canvas API key.

Direct Canvas Integration

View assignments instantly—no Canvas API key or authentication required. Just open the extension on any Canvas page.

AI-Powered Insights

Claude AI with Extended Thinking analyzes your workload and provides strategic recommendations for tackling assignments.

Weekly Schedule Generation

AI-generated 7-day study plans with time-blocked schedules optimized for your deadlines and workload.

Smart Filtering

Filter assignments by time range, course, and completion status. Grade visibility toggle for focus mode.

Structured Outputs

Consistent, reliable AI responses using Claude's latest structured output capabilities for guaranteed JSON format.

Privacy First

Canvas data stays in your browser. Only assignment info is sent to Claude AI when you request insights.

Installation

Step 1: Install Chrome Extension

  1. Download or clone the repository
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top-right corner
  4. Click "Load unpacked" and select the extension/ directory

Step 2: Configure Canvas & Claude API

  1. Click the CanvasFlow icon while on any Canvas LMS page
  2. Open Settings (gear icon) and use "Auto-Detect" to configure Canvas URL
  3. Get your Claude API key from console.anthropic.com
  4. Enter the API key in Settings for AI-powered features

Step 3: Optional - MCP Server for Claude Desktop

Connect Claude Desktop to your Canvas data for natural conversation queries.

  1. Download canvasflow-native-host.dxt
  2. Open Claude Desktop → Extensions
  3. Drag the .dxt file into the extensions area
  4. Download canvasflow-chrome-extension.zip and run the installer to configure extension ID
  5. Restart Chrome and Claude Desktop, then ask: "What are my Canvas courses?"
